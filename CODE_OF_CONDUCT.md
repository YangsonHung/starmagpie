# Code of Conduct

StarMagpie aims to maintain a friendly, professional, and sustainable open source collaboration environment.

## Expected Behavior

- Respect different experience levels, backgrounds, and viewpoints.
- Focus on the problem and provide reproducible information or concrete suggestions.
- Accept constructive feedback and keep discussions clear and restrained.
- Report risks or mistakes in a way that helps improve the project.

## Unacceptable Behavior

- Personal attacks, harassment, discrimination, or threats.
- Publishing another person's private information.
- Intentional spam, flame wars, or repeated off-topic comments.
- Sharing tokens, exploit details, or sensitive logs in public issues.

## Enforcement

Maintainers may edit, hide, or remove content that violates this code of conduct. Maintainers may also temporarily or permanently restrict participants from further interaction.

Security issues must follow [SECURITY.md](SECURITY.md).
