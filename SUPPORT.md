# Support

StarMagpie is a personal open source project and does not provide commercial support commitments.

## Getting Help

- Use GitHub Issues for bugs, feature requests, and questions.
- Search existing issues before opening a new one.
- Do not paste GitHub tokens or sensitive logs into issues.

## Maintenance Scope

Currently maintained:

- GitHub Stars sync
- SwiftData local persistence
- Search, categories, and notes
- English and Simplified Chinese localization
- Native macOS 14+ experience

Not currently promised:

- Older macOS compatibility
- Non-macOS platforms
- Commercial SLA
- Private custom features
